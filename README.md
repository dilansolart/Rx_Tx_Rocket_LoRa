# Rx_Tx_Rocket_LoRa
Sistema de Transmisión y recepción de datos Usando Modulo LoRa SX1278 para enviar datos de un cohete propulsado por agua a estación en tierra.

## Conexiones en ESP32-WROOM-32
| LoRa Ra-02 | ESP32 / Pin|
| :---------------------: | :------:|
| VCC | 3.3V |
| GND | GND |
| SCK | SCK / D18 |
| MISO | MISO / D19 |
| MOSI | MOSI / D23 |
| NSS | 5 |
| RST | 14 |
| DIO0 | 2 |

| MISO | MISO / 12 |
